# useEdictRune

Hook to edict a rune.

## Usage

```tsx
import {useEdictRune} from '@midl-xyz/midl-js-react';

function App () {
    
    const {edictRune} = useEdictRune();
    const reciever = 'bcrt1qj...'; // Ordinals address
    const amount = 1000; // Amount of rune to edict
    const runeId = 'runeId'; // Rune ID

    
    return (
        <button onClick={() => edictRune({
            reciever,
            amount,
            runeId
        })}>
            Edict Rune
        </button>
    );
}

```

## Returns

- **edictRune**: `(params: {reciever: string, amount: number, runeId: string, feeRate?: bigint}) => Promise<void>` - Function to edict a rune.
- **edictRuneAsync**: `(params: {reciever: string, amount: number, runeId: string, feeRate?: bigint}) => Promise<void>` - Function to edict a rune asynchronously.
