# useSignMessage

Hook to sign a message with the connected accounts.

## Usage

```tsx
import {useSignMessage} from '@midl-xyz/midl-js-react';

function App () {
    const {signMessage, isPending} = useSignMessage();

    return (
        <div>
            <button onClick={() => signMessage('Hello, World!')} disabled={isPending}>
                Sign Message
            </button>
        </div>
    );
}
```

## Returns

- **signMessage**: `(message: string) => Promise<string>` - Function to sign a message.
- **signMessageAsync**: `(message: string) => Promise<string>` - Function to sign a message asynchronously.
