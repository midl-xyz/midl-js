# useConnect

Hook to connect to a wallet.

## Usage

```tsx
import {useConnect} from '@midl-xyz/midl-js-react';
import {AddressPurpose} from '@midl-xyz/midl-js-core';

function App () {
    const {connect} = useConnect({
        purposes: [AddressPurpose.Payment, AddressPurpose.Ordinals],
    })

    return (
        <button onClick={connect}>Connect</button>
    )
}

```


## Returns

- **connect**: `() => void` - Function to connect to a wallet.
- **connectAsync**: `() => Promise<void>` - Function to connect to a wallet asynchronously.
- **connectors**: [`Connector[]`](/docs/glossary/types#connector) - The connectors available to connect to a wallet.
