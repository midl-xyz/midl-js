# useDisconnect

Hook to disconnect from a wallet.

## Usage

```tsx

import {useDisconnect} from '@midl-xyz/midl-js-react';

function App () {
    const {disconnect} = useDisconnect()

    return (
        <button onClick={disconnect}>Disconnect</button>
    )
}

```

## Returns

- **disconnect**: `() => void` - Function to disconnect from a wallet.
- **disconnectAsync**: `() => Promise<void>` - Function to disconnect from a wallet asynchronously.
