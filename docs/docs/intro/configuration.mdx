# Configuration

## Import

```ts
import {createConfig} from '@midl-xyz/midl-js-core';
```

## Usage

Create and export a new Midl.js config file in your project.

```ts
import {createConfig, testnet, mainnet, satsConnect} from '@midl-xyz/midl-js-core';

export const config = createConfig({
    networks: [testnet, mainnet],
    connectors: [satsConnect()]
});
```

## Parameters

#### networks

- **Type**: [`BitcoinNetwork[]`](/docs/glossary/types#bitcoinnetwork)
- **Description**: Bitcoin networks to use in the application.

#### connectors

- **Type**: [`Connector[]`](/docs/glossary/types#connector)
- **Description**: Connectors to use to connect wallets to the application.

#### chain (optional)

- **Type**: [`Chain`](/docs/glossary/types#chain)
- **Description**: The underlaying EVM chain to use in the application.


#### persist (optional)

- **Type**: `boolean`
- **Default**: `true`
- **Description**: Persist the configuration to local storage.
