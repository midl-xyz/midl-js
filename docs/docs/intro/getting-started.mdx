# Getting Started

Install Midl.js via your package manager of choice.

<NPMInstall packages={[
    '@midl-xyz/midl-js-core',
    '@midl-xyz/midl-js-react', 
   ]} 
/>


## Create config    

Create and export a new Midl.js config file in your project.

```ts
import {createConfig} from '@midl-xyz/midl-js-core';

export const config = createConfig({
    // Add your config here
});
```

## Wrap your app with Context Provider

Wrap your app with the Midl.js context provider.

```tsx
import {MidlProvider} from '@midl-xyz/midl-js-react';
import {config} from './config';

function App() {
    return (
        <MidlProvider config={config}>
            {/* Your app here */}
        </MidlProvider>
    );
}
```
