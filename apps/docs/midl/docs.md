# Overview


MIDL JS SDK is a JavaScript library that makes it easy to interact with the Bitcoin and MIDL Protocol. It provides a simple and intuitive API to interact with the blockchain, manage accounts, and sign transactions.

## Features

- **Simple API**: MIDL JS SDK provides a simple and intuitive API to interact with the blockchain.
- **TypeScript Support**: MIDL JS SDK is written in TypeScript and provides type definitions.
- **React Hooks**: MIDL JS SDK provides React hooks to interact with the blockchain in your React app.
