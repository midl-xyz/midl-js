# useToken

Get Rune by it's EVM address.

## Import

```ts
import { useToken } from "@midl-xyz/midl-js-executor-react";
```

## Example

```tsx
const { rune } = useToken(evmAddress);
```

## Parameters

| Name       | Type     | Description             |
| ---------- | -------- | ----------------------- |
| evmAddress | `string` | The EVM address of rune |
