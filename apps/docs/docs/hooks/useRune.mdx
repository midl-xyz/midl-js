# useRune

Hook to get a rune by its ID.


## Usage

```tsx
import { useRune } from '@midl-xyz/midl-js-react';

function App () {
    const { rune } = useRune({
        runeId: 'runeId',
    });

    return (
        <div>
            {rune?.symbol}: {rune?.balance}
        </div>
    );
}
```

## Returns

- **rune**: `Rune` - Rune object with its balance.
