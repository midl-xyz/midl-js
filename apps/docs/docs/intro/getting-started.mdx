# Getting Started

Install Midl.js via your package manager of choice.

<NPMInstall packages={[
    '@midl-xyz/midl-js-core',
    '@midl-xyz/midl-js-react', 
    '@midl-xyz/midl-js-executor'
   ]} 
/>

:::note
For using executor layer also install patched viem

```js
// package.json dependencies
 "viem": "npm:@midl-xyz/midl-viem@2.21.36",
```
:::

## Create config    

Create and export a new Midl.js config file in your project.

```ts
import {createConfig, regtest, leather} from '@midl-xyz/midl-js-core';

export const config = createConfig({
    networks: [regtest],
    connectors: [leather()],
    persist: true
});
```

## Wrap your app with Context Provider

Wrap your app with the Midl.js context provider.

```tsx
import {MidlProvider} from '@midl-xyz/midl-js-react';
import {config} from './config';
import {WagmiMidlProvider} from '@midl-xyz/midl-js-executor';


function App() {
    return (
        <WagmiProvider
            config={wagmiConfig}>
            <MidlProvider config={midlConfig}>
                <QueryClientProvider client={queryClient}>         
                    <WagmiMidlProvider />
                    {children}
                </QueryClientProvider>
            </MidlProvider>
        </WagmiProvider>
    );
}
```

:::warning
Only Next.js higher than 15.0.2 is supported. You should enable async WebAssembly support in your `next.config.js` file.

```js
/**
 * @type {import('next').NextConfig}
 */
const config = {
  webpack(config) {
    config.experiments = {
      ...(config.experiments || {}),
      asyncWebAssembly: true,
    };

    return config;
  },
};

module.exports = config;
```

:::


## Use Midl.js hooks

Use Midl.js hooks to interact with the blockchain.

```tsx
import {useConnect, useAccounts} from '@midl-xyz/midl-js-react';

function Page () {
    const {connect, connectors} = useConnect();
    const {accounts, isConnected} = useAccounts();

    return (
        <div>
            {connectors.map(connector => (
                <button key={connector.name} onClick={() => connect(connector.name)}>
                    {connector.name}
                </button>
            ))}

            {isConnected && (
                <div>
                    <h2>Accounts</h2>
                    <ul>
                        {accounts.map(account => (
                            <li key={account}>{account}</li>
                        ))}
                    </ul>
                </div>
            )}
        </div>
    );
}
```
