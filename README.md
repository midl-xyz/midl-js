### Features

- **Connect to any Bitcoin network** (mainnet, testnet, regtest, etc.)
- **Publish transactions** to the Bitcoin network
- **Sign PSBTs**
- **Sign messages** using BIP322 and ECDSA
- **Transfer Runes**
- **Etch runes** on the blockchain
